# Company Logos

Place default company logos here with naming convention:
- `company-name.png` or `company-name.jpg`
- Use lowercase with hyphens for spaces
- Examples: `rakamin-academy.png`, `gojek.png`, `tokopedia.png`

## Usage in Components:
```typescript
import RakaminLogo from '@/assets/company-logos/rakamin-academy.png';
```
